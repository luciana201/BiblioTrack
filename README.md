## Características

### Tipos de Publicaciones
- **Cómics**: Con ilustrador, volumen y demografía
- **Novelas**: Con tipo de narrador y número de páginas
- **Libros Técnicos**: Con tema y nivel de dificultad

### Gestión de los Usuarios
- Crear usuarios con un ID unico.
- Buscar usuarios por ID.

### Sistema de Reseñas
- Calificación de 1 a 10
- Registro automático de fecha

### Estados de Lectura
- **PENDIENTE**: Aún sin leer
- **LEYENDO**: En proceso 
- **LEIDO**: Completado
